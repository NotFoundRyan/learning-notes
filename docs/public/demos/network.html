🌐 UDP/TCP 网络通信演示

UDP vs TCP 对比

UDP 用户数据报协议

  • ⚡ 无连接，快速传输
  • 📦 数据包独立，无序
  • ❌ 不保证可靠性
  • 📡 支持广播/组播
  • 💡 适用: 实时音视频、DNS

TCP 传输控制协议

  • 🔗 面向连接，三次握手
  • ✅ 可靠传输，有序
  • 🔄 自动重传机制
  • 📊 流量/拥塞控制
  • 💡 适用: HTTP、文件传输
