UDP vs TCP 协议对比

协议特性对比

UDP 用户数据报协议

  • ⚡ 无连接，快速传输
  • 📦 数据包独立，无序
  • ❌ 不保证可靠性
  • 📡 支持广播/组播
  • 💡 适用: 实时音视频、DNS

TCP 传输控制协议

  • 🔗 面向连接，三次握手
  • ✅ 可靠传输，有序
  • 🔄 自动重传机制
  • 📊 流量/拥塞控制
  • 💡 适用: HTTP、文件传输
特性 UDP TCP
连接方式 无连接 面向连接
可靠性 不保证送达 保证送达
顺序保证 可能乱序 保证顺序
传输速度 快 相对慢
资源消耗 低 高
首部开销 8 字节 20 字节
广播/组播 支持 不支持

应用场景选择

⚡ UDP 适用场景

  • 实时音视频通话
  • 在线游戏
  • DNS 查询
  • 局域网设备发现
  • 传感器数据上报

✅ TCP 适用场景

  • 网页浏览 (HTTP)
  • 文件传输 (FTP)
  • 邮件发送 (SMTP)
  • 远程登录 (SSH)
  • 数据库连接
