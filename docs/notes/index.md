# 学习笔记

欢迎来到我的学习笔记！这里记录了我的学习过程和心得体会。

## 目录结构

```
docs/
├── notes/          # Markdown 笔记
├── demos/          # 演示文档
├── public/
│   ├── images/     # 图片资源
│   └── demos/      # HTML 演示文件
└── .vitepress/     # VitePress 配置
```

## 快速开始

1. 在 `docs/notes/` 目录下创建新的 `.md` 文件
2. 在 `docs/public/images/` 目录下放置图片
3. 使用相对路径引用图片：`![图片描述](/images/xxx.png)`
